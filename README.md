## 📌 Project Overview

This project performs **Sentiment Analysis on Social Media data** using a hybrid approach that combines **BERT (Bidirectional Encoder Representations from Transformers)** for feature extraction and **XGBoost** for sentiment classification.

The system automatically classifies social media posts into:

- 😊 **Positive**
- 😐 **Neutral**
- 😡 **Negative**

This helps organizations analyze **public opinion, customer feedback, and brand perception** from social media platforms.

---

## 🎯 Objectives

- Analyze large amounts of social media text
- Extract contextual embeddings using **BERT**
- Classify sentiments using **XGBoost**
- Evaluate model performance using standard ML metrics
